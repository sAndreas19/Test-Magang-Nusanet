# Nusanet Shift Scheduler (Internship Test)
Aplikasi perhitungan siklus jadwal shift karyawan dengan pola rotasi dinamis, dilengkapi dashboard visual dan export Excel.

## Requirements
Pastikan komputer Anda sudah terinstall:
* **Node.js** (Versi LTS, disarankan v18 ke atas)
* **NPM** (Bawaan Node.js)

## âš™ï¸ Cara Install
1. **Clone** repository ini ke komputer Anda (atau extract file ZIP).
2. Buka terminal di dalam folder project.
3. Jalankan perintah berikut untuk menginstall semua dependencies:
   npm install

## Cara Menjalankan Aplikasi
Start server dengan perintah: <br>
node index.js
Tunggu hingga muncul pesan: API Server berjalan di http://localhost:3000

Buka browser dan kunjungi alamat: http://localhost:3000 atau http://127.0.0.1:3000
